CSS Examples - Panels

Panels can be used to visually section off related content elements. They are similar to "articles", except that they visually emphasize content.

Generally speaking, a panel is comprised of a div with the "panel" class. It can have up to three expected direct child types:

  • "header" or "head"
  • "content" or "body"
  • "footer" or "foot"

At least one of the child types is expected to be present. Up to all three can be present. And ordering matters. That is, if present, the header child is expected to be first, and the footer child is expected to be last.

Direct children of the "header" and "footer" sections will always try to space out evenly, while direct children in the "content" section will format in normal paragraph format.

Tabbed Panel Examples

Panels with clickable tabs to show/hide content.

To create, start with a standard panel and add the class "tabs" to the outermost div. Then within the panel's body/content section, create divs with the class "tab" for each tab you want. The first header element within each tab will become the clickable nav text.
