CSS Examples - Text Highlighting

These are various forms of emphasizing/highlighting text. These only affect the direct text and not the content the text is contained within.

To only change the color of the text, add the desired class (primary, info, etc) to the <p> or <span> element. To also change the background of the text, add the "highlight" class.
