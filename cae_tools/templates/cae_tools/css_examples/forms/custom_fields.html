Custom "Select Button"-Related Form Fields

The "Select Button" widget is selected by using the SelectButtonsWidget from the cae_home forms.py file.

The "Select Button" form widgets allow a choice field that looks and acts like clickable buttons. Currently, implementation only allows one choice to be selected at a time with this widget. Thus it should not be used with a "MultipleChoiceField" for now.

Note that these buttons can also be toggled to display on the side, next to all other fields in the form. This is done by simply using the SelectButtonsSideWidget widget. Other than displaying on the side of the form, this should behave identically to the standard Select Buttons widget.

Custom Select2-Related Form Fields

The "Select2" widget is our Django implementation of https://select2.org/.

In short, it's an attempt to have more dynamic and universal select choices across all websites, languages, and browsers. Our implementation here is only a basic one, with a lot of potential functionality that can be imported/added. See select2 official docs for full details of what it's meant to do.
