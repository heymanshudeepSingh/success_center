Idk if they're useful to have as "generic elements in CaeHome, but they're here now.

Select-Square Elements

Clickable selection boxes to show/hide content.

These are somewhat similar to our Tabbed Panels Element as well as our Select Scroll Element.


Each div directly in the class root will be considered a separate square. The first child in each div will be set as the value displayed in the select square. All other content will be hidden until the given square is clicked by the user.

Select-Scroll Elements

Clickable selection scrolls to show/hide content.

These are somewhat similar to our Tabbed Panels Element as well as our Select Squares Element.


Each div directly in the class root will be considered a separate scroll-box. The first p child in each div will be set as the value displayed in the scroll-box. All other content will be hidden until the given scroll is clicked by the user.
