CSS Examples - Articles

Articles are good elements to split up larger sections of content, while having minimal visual separation of said content. They are similar to "panels", except that they visually de-emphasize content.

To create an article, one can either use the <article> element or a div with the class of "article".

Basic Use of Sections within Articles

Usually, to subdivide content within an article, you would use a section element, like this one. To create an section, one can either use the <section> element or a div with the class of "section".

Similarly to articles, sections can add the "color" class to get a visible border and background.
