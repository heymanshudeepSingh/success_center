CSS Examples - Articles

Article Examples

Articles are good elements to split up larger sections of content. By default, they are clear. To create an article, one can either use the <article> element or a div with the class of "article".

But adding the "color" class gives articles a border and background, like this. Or if using a div element, then use the "article-color" class.

Usually, to subdivide content within an article, you would use a section element, like this one. Again, to create an article, one can either use the <section> element or a div with the class of "section".

Similarly to articles, sections can add the "color" class to get a visible border and background. If using a div element, then use the "section-color" class.
