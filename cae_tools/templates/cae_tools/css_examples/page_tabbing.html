CSS Examples - Page Tabbing

"Page tabbing" elements separate sections of a page into unique views using JavaScript. These sections can be shown or hidden based on user click actions.


How it Works:

Clickable tabbings are created for all divs with the "page-tabs" styling. There will always be an "All" tab that shows all content. This is what shows on page load. Clicking any other tab will hide all other content and show only content for that tab. Tabs are determined by "direct child" H2 elements of this "page-tabs" div. Content is associated with a tab if it is directly beneath the corresponding H2 element.


If headers are not specified, then all "direct child" H2 elements are turned into tabs. Otherwise, headers can be specified in the "page-tabs" divs element, using a "data-headers" attribute. Any headers not included in "data-headers" is lumped into an "Other" tab. Note that these headers must be separated by a space and comma.
