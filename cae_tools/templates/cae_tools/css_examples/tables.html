CSS Examples - Tables

Table Examples

To comply with screen reader standards, tables should only be used when displaying data that makes sense in a row column format. Basically data that would make sense in a spreadsheet.

Be careful with table width and mobile screens. Always make sure to test your page output on mobile devices if it contains a table. If the table is too wide, consider breaking it up into multiple tables, if possible.
