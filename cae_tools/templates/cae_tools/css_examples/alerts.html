CSS Examples - Alerts

Alerts can be used anywhere on the page. They are generally intended to pop up as a notification to user within context of the page, such as to display a form error.

Alert elements can be created by specifying a div with a class of "alert". Direct child elements placed inside the alert will automatically be spaced out as evenly as possible.
