CSS Examples - Status Messages

Status Messages are similar to alerts, but intended to show up at the top of the page, usually after something like a form submission.

Generally speaking, "Django messages" should be used to automatically create these. If you need to to create something like this in the middle of the page, more often than not, an alert element should be used instead.

These will have an "x" on the right hand side for the user to remove the message from the page.
