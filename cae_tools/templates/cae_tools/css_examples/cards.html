CSS Examples - Card

Cards are meant to be used similarly to panels, but generally more to describe a single entity, such as how we use them to display Applications in the GradApps project.

Unlike panels, cards are less customizable. A card is meant to have three direct children:

  • "header" or "head"

  • "body"

  • "toggle"
