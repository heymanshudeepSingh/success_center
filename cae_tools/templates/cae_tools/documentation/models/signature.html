Handling Signature Forms & Signature Models

See also generic form implementation at Custom Form Field Examples.


CAE Workspace provides a generic implementation of signature handling, for such as when a user uses a tablet or touch pad to physically write their signature.

When logic supports it, we recommend not storing the signature to the database (for security). And instead, simply using the generic SignatureWidget form field (from cae_home/forms.py) to get a signature and then process it directly in the view, without saving it long-term.

However, this is not necessarily universally applicable and some logic requires saving long-term, in which case you should inherit signature logic from cae_tools/models.py. Example below.
