Provided JavaScript Libraries & Logic

Workspace provides some JavaScript libraries by default. These are automatically loaded and available to all templates that extend logic from CAE Home.


Third Party Libraries

  • JQuery - JQuery is a library that adds a significant amount of general logic and helper functions to the base JavaScript language.

    In a lot of cases, JQuery accomplishes what JavaScript could already do, but in far fewer lines and much easier to maintain code. Plus, according to the documentation, it has minimal footprint, and is cross-browser compliant.

  • React - React is a library that overhauls JavaScript to be much more dynamic and organized for large-scale pages.

    Note that React is a pretty significant overhaul of JavaScript, on its own. To the point that it can act as a full, stand-alone web server. Workspace only uses it for front-end management, but there's still a bit of a learning curve to getting it working.

  • Flatpickr - Flatpickr is a third-party, JavaScript-based datetime form widget. It's used for some of Workspace's Custom Datetime Form Widgets.

  • Select2 - Select2 is a third-party, JQuery-based selectbox form widget. It's used for some of Workspace's Custom Select Form Widgets.


For more info on differences between JavaScript, JQuery, and React (as well as when to use which), see documentation at <project_root>/documents/javascript_and_react.md.

Custom JavaScript Logic

CAE Workspace also provides a handful of custom JavaScript logic. This includes things like mobile handling, custom form widgets, custom interactive CSS elements, and more.

Most of these are showcased in the CSS Example pages. The associated code is located at <project_root>/cae_home/static/cae_home/js/.
