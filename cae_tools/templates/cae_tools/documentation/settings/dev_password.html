Dev Passwords

When the project generates seeds, there is a "dev password" that is used for all seeded user accounts. This password is defined towards the bottom of the Local Environment Settings file, so that it can be changed locally on a per-machine basis.

Password Values

The default value for this password is temppass2.

Since this is stored in locally in plaintext, if you change it, it's recommended to change it to something easy to remember that is not a normal password you use for any other account logins.

If desired, it can be something really insecure if you want, such as a single character. It only affects your local development environment so there's really no harm in making it whatever you want.

To create a new user account that's generated via seeds, edit the file located at /cae_home/commands/seeders/user.py. See the seeding documentation for more.
