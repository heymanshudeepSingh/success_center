Project Logging

The workspace project has custom logging logic, which saves all console output to file.

The default location for logs is /settings/local_env/logs/ and can be changed in the Local Environment Settings.

This log file is effectively the same as logging in most CAE Center Python projects, but with a few extra filters and handlers for web serving.
