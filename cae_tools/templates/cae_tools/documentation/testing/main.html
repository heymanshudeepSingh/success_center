    Documentation - Custom Workspace "Testing" Logic

    CAE Workspace makes use of standard Django UnitTests for a majority of testing. For any tests that require mock-server instances, we use Selenium.

    In either case, there are custom utility classes in cae_home/tests/utils.py, in order to make testing a little bit easier to write. These utility test classes inherit from the standard from django.test import TestCase, so they have all the original functionality that Django provides, as well as the custom Workspace additions.
