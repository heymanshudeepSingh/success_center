    Color Tool Current State

    Originally (back when Workspace was in development, around 2019 or so), this page was thrown together to help quickly and dynamically compare and select colors, when working with front-end css.

    Currently, this page has been mostly-ignored, and some functionality/styling has broken. It still somewhat-works, but there are professionally-made pages that do much more.

    Functionality

    This page shows various colors, all in clean little easily-selectable boxes. Each box displays the same color value in formats of HEX, RGB, and HSL.

    Clicking any of these colors will "select" it, allowing for further examination and manipulation.

    Rows:

    Top Row - WMU's "official" colors.

    Second Row - We decided that some of WMU's colors were a bit of an eyesore on an actual website, so we have toned-down (but similar) colors on the second row.

    The bottom two rows allow quickly selecting a change in saturation and lightness, for the currently selected color.
