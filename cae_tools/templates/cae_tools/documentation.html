    Custom Project Logic Documentation

    We opted to put project documentation here to ensure that everyone who has access to CAE Workspace is able to access this documentation.

    Here, we describe all the various ways in which the CAE Workspace project is unique compared to a standard Django project. This includes all the provided additional and custom functionality, as well as notes of related quirks.


    Note: Django does provide DjangoDocs as a reference for using the Django framework in general. However, it can be a bit verbose and confusing for those new to the Django framework.

    For more simplified examples and logic that is standard to Django itself, see these GoogleDocs powerpoints.
